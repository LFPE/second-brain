# 📚 Aprendizados — Projeto Tracker

> Registro de conhecimentos adquiridos durante o desenvolvimento do projeto **Tracker**.

---

## Integração com API do Excel

Durante o desenvolvimento do Tracker foi necessário aprender como integrar o sistema com a API do Excel para manipulação automática de planilhas.

### Conceitos aprendidos

- Comunicação com APIs externas
- Envio e recebimento de dados
- Estrutura de requisições
- Tratamento de respostas da API

### Aplicação no projeto

A API foi utilizada para:

- Criar planilhas automaticamente
- Organizar dados recebidos
- Automatizar processos que antes eram manuais

### Resultado

O sistema consegue gerar planilhas estruturadas automaticamente a partir de dados brutos.

---

## Automação de Dados

Um dos principais objetivos do Tracker foi automatizar processos de organização de dados pois no meu trabalho tinha que analisar e criar muitos relatorios diarios com esse trabalho massante tive a ideia de criar um sistema.

### Conceitos aprendidos

- Processamento de dados
- Estruturação de informações
- Separação automática de dados
- Limpeza de dados
- Api

### Aplicação no projeto

O sistema foi desenvolvido para:

- Receber dados brutos
- Processar automaticamente as informações
- Organizar os dados em estruturas claras 
- Criação de relatorios via Excel

### Resultado

Redução significativa de trabalho manual na organização de dados das Planilhas.
